# player-performance-prediction-ml
Machine Learning project demonstrating Ridge, Lasso, ElasticNet, PCA and Cross Validation for overfitting control using a cricket performance prediction dataset.
# Player Performance Prediction ML

This project demonstrates how to control overfitting in Machine Learning using:

- Ridge Regression
- Lasso Regression
- ElasticNet
- PCA (Principal Component Analysis)
- Cross Validation

## Project Goal

Predict cricket player performance using regression models and improve generalization using regularization techniques.

## Techniques Used

- Linear Regression
- Ridge
- Lasso
- ElasticNet
- PCA
- Cross Validation

## Tech Stack

- Python
- Pandas
- Scikit-learn
- NumPy

## Dataset

Custom cricket performance dataset with features like:

- last 5 match runs
- average
- strike rate
- pitch score
- weather score
- fitness score

Target:
runs_next_match
